The Neural Dendrite Tree

The sculpture grows upward like an inverted neural tree. A central trunk (brainstem) carries power and signals. Branches spread outward at each layer — input dendrites at the bottom, hidden processing in the middle canopy, output axons reaching up top.

Each branch terminus is a mechanism cluster (gears, cams, logic gates). Signals flow as marbles/balls rolling along the branches. The Galton board sits at the base (brainstem), probability cascading down into the root system.

  • Natural metaphor: growth, branching, reaching
  • Gravity assists signal flow downward (backprop)
  • Each branch is an independent subsystem
  • Galton board in the trunk/root system
  • ~1.5m tall, ~1.2m spread

The Cortical Layer Stack

Horizontal floating platforms stacked vertically, each representing a network layer. Input tokens enter at the top, signals cascade downward through attention, hidden state, computation, to output at the bottom.

Each platform is an organic, rounded slab (like a brain cross-section) with mechanisms mounted on top. Platforms connected by vertical signal pathways — strings, rods, marble tracks running between layers.

  • Clear layer separation = easy to understand
  • Gravity drives forward pass (top to bottom)
  • Each platform independently removable/serviceable
  • Galton board spans multiple layers vertically
  • ~1.5m tall, ~1.2m wide platforms

The Synapse Constellation

No box, no layers — mechanism clusters float in space like neurons, connected by organic curved pathways (rods, tubes, marble tracks, strings). The topology IS the sculpture. A large central node houses the core computation (ALU, logic gates). Satellite nodes handle specialized functions.

Tendrils extend outward like dendrites — some carry signals, others are purely sculptural. The whole structure hangs from a minimal armature or mounts to a wall, giving a floating, neural appearance.

  • Most organic and brain-like of all options
  • Each node is a self-contained mechanism module
  • Connections ARE the visual drama (marbles rolling along curved tracks)
  • Galton board as a vertical cascade between two nodes
  • ~1.5m spread, irregular organic footprint

The Split-Brain Machine

Two hemispheres — left brain = training (Act 1), right brain = inference (Act 2) — connected by a central spine that houses the Galton board. The corpus callosum (cross-connections) carries shared weights between hemispheres.

This maps directly to your two-act narrative. During training, the left hemisphere is alive with motion. When training completes, activity shifts to the right. The Galton board in the spine shows probability throughout both acts.

  • Two-act narrative maps to two physical hemispheres
  • Galton board IS the brainstem (probability flows down the spine)
  • Corpus callosum = weight sharing (visible cross-connections)
  • Bilateral symmetry is instantly brain-like
  • ~1.5m wide, ~1.2m tall, bilateral layout
