The Weighted Archimedes Lever

Neural operation: Weighted Sum + Backpropagation

How It Works

A brass beam (150-200mm long) balanced on a central fulcrum. Along the beam, sliding brass blocks sit in dovetail slots. Each block's position along the beam encodes one weight value.

Active inputs push brass rods upward through the midplane. The rods contact the beam at the corresponding weight positions. The beam tips proportional to the weighted sum of active inputs.

Why This Is Brilliant

  • Same lever does forward AND backward pass
  • Only 2 friction points (fulcrum + beam)
  • Weight values are literally visible (block position)
  • ReLU = simple mechanical stop at zero
"The beam tilts like a thinking scale. During training, you watch the brass blocks slowly creep along the beam — the weights physically moving as the machine learns."
Reuleaux Set T Margolin aesthetic

Quantities

6 levers total: 3 hidden neurons + 3 output neurons

2

Self-Locking Worm Gear Grid

Neural operation: Weight Storage (Persistent Memory)

How It Works

Each worm gear cell: a small worm (screw) meshes with a worm wheel. The wheel's rotation positions a sliding block on the Archimedes lever. Because the worm's friction angle exceeds its lead angle, the wheel cannot back-drive the worm.

This means: once a weight is set, it stays put until actively driven. Power off the motor — weights remain. This IS persistent memory.

Weight Update Mechanism

During backprop, an error signal drives a ratchet that turns the worm by discrete clicks. Each click shifts the lever block by a fixed increment. This is mechanical stochastic gradient descent with a fixed learning rate.

Visual Impact

"42 worm gears arranged in a 7×6 grid, brass on black. During training, you see individual worms turn — click, click — as the machine adjusts its beliefs. Gears near important features spin more. Irrelevant weights stay still."
Reuleaux Set M Torsen heritage

Quantities & Scale

42 cells. Each cell: ~15mm worm + 12mm wheel. Grid: ~120mm × 100mm total.

3

The Whippletree (Calder Mobile)

Neural operation: Loss Computation

How It Works

Three sub-beams compare each output neuron's value against the target from the pin drum. The difference (error) tilts the beam. A top-level beam aggregates all three errors into total loss.

When the network is wrong, beams tilt dramatically. As it learns, they gracefully settle toward level. An eddy-current damper (copper plate + permanent magnet) makes the settling motion slow and beautiful — like a Calder mobile finding equilibrium.

Why Whippletree

  • Zero friction in the computation (just pivots in air)
  • Zero gears, zero strings — pure lever physics
  • Error magnitude = beam tilt angle — instantly readable
  • Historical: whippletrees distributed horse team forces evenly
"A mobile of brass and steel. During training it lurches, recovers, overshoots, settles. Over epochs, the oscillations dampen. When the machine finally 'gets it,' the whole assembly hangs still. Silence = intelligence."
Reuleaux Set T Calder mobile zero-friction computation
4

The Clock Escapement (Heartbeat)

Neural operation: Sequencing / Timing

How It Works

A clock escapement on the main motor shaft converts continuous rotation into discrete, evenly-spaced ticks. Each tick advances the computation by one neuron.

The escapement doesn't just time the machine — it creates the rhythm. The machine doesn't compute smoothly; it thinks in beats, like a brain firing neurons in sequence.

Why Not a Governor

  • Governor = continuous, smooth, boring
  • Escapement = discrete, rhythmic, alive
  • Each tick is audible — you can hear the machine thinking
  • Speed adjustable by changing pendulum/balance length
"The most clock-like sound in the sculpture. Regular, meditative, almost breathing. Speeds up slightly under no-load (correct predictions) and labors under heavy error (wrong predictions). The machine literally sounds stressed when it's confused."
Reuleaux Set X audible computation

The Brachistochrone Ball Track

Neural operation: Gradient Descent Visualization

How It Works

A brass ball sits in a cycloid-shaped track. The ball's position along the track is mechanically linked to the total loss from the whippletree. High loss = ball at the top. As the machine learns, the ball rolls down the curve of fastest descent.

This is not a metaphor. The brachistochrone IS gradient descent — mathematically isomorphic. Bernoulli solved it in 1696. Now it's a physical sculpture element.

Visual Impact

  • Early training: ball high up, rolling fast (steep gradient)
  • Mid training: ball in the curve, decelerating
  • Converged: ball rests at bottom, barely moving
  • The curve shape itself teaches calculus of variations
"A single brass ball rolling down a golden curve. It starts fast — the machine is learning rapidly. Then it slows. And slows. And finally... rests. The machine has learned. The ball is still."
Reuleaux Set R Bernoulli 1696
6

The Training Data Pin Drum

Neural operation: Training Data Storage + Epoch Cycling

How It Works

A transparent cylindrical drum sits elevated at the top of the sculpture. Brass pins encode the input data; red pins encode the correct answers (targets). The drum rotates continuously during training, feeding one example per escapement cycle.

Modular Pin Strips

Pin strips slide in/out of the drum. Change the curriculum by swapping strips — physically reprogrammable, like changing a punch card. Different strips = different training data = different learned behavior.

Epoch Counter

A worm-and-wheel on the drum shaft counts complete revolutions. Pointer dial shows: "The machine has seen all training data N times." Convergence typically happens around epoch 15-30.

"The drum rotates above everything, like a music box cylinder. Brass and red pins catch the light. Below it, the whole machine responds — levers tilting, worms clicking, beams swaying. The drum is the teacher, the machine is the student."
Reuleaux Set A Jacquard heritage
7

Dog Clutch Mode Switch + Answer Prism

Neural operation: TRAIN / PREDICT mode + Output Display

Dog Clutch

A physical lever with a satisfying clunk. In TRAIN position, the pin drum drives the input selectors automatically. In PREDICT position, the drum disengages and 3 manual dials connect to the selectors instead.

The viewer literally switches the machine between "learning mode" and "thinking mode."

Answer Prism

A 3-sided triangular prism with one word engraved on each face. Driven by a detented rack-and-pinion from the output neuron with the highest activation. The prism rotates through positions with distinct clicks — building suspense — then locks at the prediction.

Unlike a dial gauge (where all answers are visible), the prism only shows one face at a time. Mystery preserved.

"You turn the three input dials: Cat... Chases... Seed. The machine ticks six times. The prism rotates: click... click... CLUNK. 'Cat' faces you. The machine has never seen this combination before. It generalized."
Reuleaux Set A + N
