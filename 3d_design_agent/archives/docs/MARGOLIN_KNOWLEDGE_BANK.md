## Core Philosophy
1. **Waves as fundamental principle**: Every sculpture is a physical manifestation of wave mathematics
2. **Analog computation**: The mechanism IS the computer. Sprocket ratios = frequency ratios. Cam profiles = waveforms. Phase offsets = physical spacing along a shaft
3. **One motor when possible**: Single input → complex emergent behavior through mechanical advantage and wave superposition
4. **String as universal transmission**: Cable/string is the primary force-transmission medium — lightweight, flexible routing, can go around corners via pulleys
5. **Friction as the enemy**: After ~9 pulleys in series, friction force ≈ 10× payload weight. Every design fights this limit
6. **Beauty through mathematics**: Not decoration — the wave equation itself generates the aesthetic. When the math is right, the motion is beautiful

## The Central Equation
Every Margolin wave sculpture implements some form of:

```
h(x, y, t) = Σ Aᵢ · sin(kᵢ · dᵢ(x,y) - ωᵢt + φᵢ)
```

Where:
- `Aᵢ` = amplitude of wave component i (set by cam eccentricity or helix radius)
- `kᵢ` = wave number (spatial frequency)
- `dᵢ(x,y)` = distance function from wave source i to point (x,y)
- `ωᵢ` = angular frequency (set by sprocket ratio to motor)
- `φᵢ` = phase offset (set by physical position along cam/helix shaft)

The mechanism's job is to evaluate this equation at every point in the grid, every moment in time, using only rotating shafts, strings, and gravity.

## 8 Mechanism Families (Taxonomy)

### Family 1: Camshaft-Driven
**Principle**: Rotating shaft with offset disc cams. Each cam's angular position along shaft = spatial phase. Cam eccentricity = amplitude.
**Key Example**: Square Wave — two camshafts perpendicular, 9 plywood circles each offset 45°. Formula: `A·sin(a) + B·sin(b) + C`
**Variable Amplitude Innovation**: Sliding the plywood disc more/less centered on the shaft changes eccentricity = changes amplitude dynamically
**Also**: Magic Wave (9 motors, 3000 pulleys), Confluence (2024, with Python-computed string paths)

### Family 2: Helix-Driven
**Principle**: Rotating helix shaft acts as continuous cam. Position along shaft = continuous phase. Cable wraps around helix, connects to slider/block.
**Key Example**: Triple Helix — 3 helix shafts at 120°, each drives tier of sliders through steel cable. Block height = sum of 3 slider displacements.
**Also**: Anemone (2 wooden helices, 48 strings), Helix, Dandelion Wave (geodesic helix)

### Family 3: Eccentric Cam / Ring Drive
**Principle**: Eccentric ring or cam provides simple harmonic motion. Often combined with string routing for distribution.
**Key Example**: River Loom — 2 eccentric cams through pentagonal web, 271 strings (prime number avoids visual repetition)
**Also**: Helio Curve (ground-level eccentric drives)

### Family 4: String-Weave / Loom
**Principle**: Strings routed through matrix of pulleys/blocks. The weave pattern determines how input motion maps to output motion at each point.
**Key Example**: Confluence (2024) — Python+matplotlib computed optimal string paths. For n pulleys, 2ⁿ possible paths; shortest = correct.
**Also**: Interlaced (overhead flat weave), Reflected Wave (string-through inversion), Cadence (hex matrix)

### Family 5: Multi-Frequency Fourier
**Principle**: Integer-ratio sprocket chains maintain exact frequency relationships between multiple wave components. Motion = Fourier series sum.
**Key Example**: Fourier Caterpillar — 3 frequency components decomposed from caterpillar gait, integer-ratio sprockets, tapered pulleys, moving+static trucks
**Also**: Arc Line (4 sprocket sizes: 20,21,27,35 teeth → 27-minute cycle from single motor)

### Family 6: Topology + Waves
**Principle**: Wave equation mapped onto non-Euclidean surfaces. Mathematical constraint: waveform must be continuous on the surface.
**Key Example**: Mobius Wave — 3.5 wavelengths (NOT integer, to avoid cancellation at twist). Edges of Mobius strip lie on a torus → 3D parametric equations. Steel strip cut flat from sheet, bent into torus shape using wooden jig.
**Also**: Trefoil (trefoil knot = hypotrochoid path), Dandelion Wave (sphere with 132 geodesic division points)

### Family 7: Epicycloid / Parametric Path
**Principle**: Nested rotating systems trace complex paths. Sprocket ratio combinations explored computationally, built mechanically.
**Key Example**: Arc Line — each ring moves in 2D plane but the chain of 20 rings appears to trace 3D paths. Brain assumes minimum curvature → infers depth. The perceived depth IS the imaginary component of a complex wave. 4 axes driven by single rotation.
**Also**: Trefoil (3 cables trace hypotrochoid)

### Family 8: Interactive / Performance
**Principle**: Human bodies as wave generators. Dancer motion → string → sculpture response.
**Key Example**: Connected — 35'×10'×20', 88 pulleys, 1 motor + human dancers. Paper, magnets, string, steel.

## Deep Mechanism Details (From Photo Analysis)

### Square Wave Camshaft Mechanism (PHOTO DETAIL)
- **Overhead mechanism visible**: Two horizontal shafts perpendicular to each other
- **Each shaft carries 9 plywood disc cams** sequentially offset by 45°
- **Disc cams are large plywood circles** — NOT precision machined. Hand-cut on bandsaw
- **Variable amplitude**: Each disc can SLIDE along the shaft to change eccentricity (distance from center to shaft axis). More offset = larger amplitude wave
- **Strings descend from cam followers** down to translucent grid of wooden slats below
- **The two perpendicular camshafts** create two independent wave directions that superpose in the grid

### Triple Helix Construction Sequence (11 PHOTOS ANALYZED)
1. **Polycarbonate matrix**: Sheets being drilled and routed with precision hole patterns for string routing
2. **Aluminum helix discs**: Stack of machined aluminum discs with central bore — these are the helix segments
3. **Machined collars**: White aluminum collars that connect helix segments, creating the continuous spiral
4. **Steel triangular truss frame**: Welded steel triangles forming the support structure
5. **Basswood hex blocks**: Individual hexagonal blocks, each with string attachment point
6. **Assembly from below**: View showing the helix mechanisms with white aluminum collars, strings descending through matrix to blocks below

### Dandelion Wave Pulley System (PHOTO DETAIL)
- **Geodesic framework**: Radial metal rods from central hub, like a geodesic dome skeleton
- **1,584 pulleys total**: Dozens of 6-spoke wooden pulleys visible in two colors (natural wood + teal/blue)
- **Pulley function**: Each pulley redirects a string. The two colors likely represent two different wave components
- **Hub**: Central metal hub where all radial rods converge — motor drive enters here
- **132 division points**: Vertices of the geodesic grid, each an output point

### Arc Line Sprocket Mechanism (PHOTO DETAIL)
- **Overhead golden wooden assembly**: 4 different-sized sprockets connected by chains
- **Off-the-shelf sprockets**: 20, 21, 27, 35 teeth — chosen for the specific frequency ratios they create
- **Strings fan down**: From each sprocket assembly point, strings descend to the 20 steel rings below
- **Rings**: Form a chain-like arc at the bottom, connected by strings to create coupled motion
- **27-minute complete cycle**: The LCD of the 4 sprocket ratios determines full cycle length

### Anemone Mechanism (PHOTO DETAIL)
- **Central cylindrical hub**: Possibly a cam drum or bearing assembly at the core
- **Pink vertical rods**: 48 elements fanning outward radially
- **Blue elements**: 48 steel bars connecting helices to upper rods
- **Two wooden helices near ground**: Turned on Margolin's lathe (biggest turning he ever made)
- **Legs from River Loom**: Curved steel legs repurposed from previous sculpture

### Mobius Wave Construction (FROM TEXT)
- **Key insight**: Edges of a Mobius strip lie on a torus
- **3D parametric equations** describe the shape
- **Fabrication**: Solved torus shape flat → cut from steel sheet with jigsaw → drilled holes while flat → bent into 3D using wooden jig
- **Wave mechanism**: Each cherry wood piece pulled by string one direction, pair of springs pulls back (spring return, not gravity)
- **3.5 wavelengths**: Half-integer specifically chosen so wave doesn't cancel at the twist point

### Reflected Wave / Interlaced Pair (PHOTO + TEXT)
- **Sister pieces**, both 2019, both almost 2D (flat wall-mounted)
- **Interlaced**: 180 wood pieces, 2 motors, overhead flat weave drives two waveforms that pass THROUGH each other. Alternating slats belong to different wave groups
- **Reflected Wave**: Strings pass through upper blocks → pull lower blocks in OPPOSITE direction. Creates mirror-image wave pair. Space between the two wave surfaces is the sculptural form

## Key Mathematical Concepts in Physical Form

### 1. Wave Superposition (Mechanical Summation)
**Physical implementation**: Each hanging block's height = algebraic sum of multiple wave components.
- Triple Helix: block contacts 3 sliders (one per helix tier), each at different phase → height = sin(a) + sin(b) + sin(c)
- Square Wave: two perpendicular camshafts → each point gets contribution from both → h = A·sin(x) + B·sin(y)
- Cadence: single motor through hex matrix with 120° maple links → superposition through geometry

### 2. Fourier Decomposition (Sprocket Ratios)
**Physical implementation**: Integer-ratio sprockets maintain exact frequency relationships.
- Fourier Caterpillar: 3 components sufficient for caterpillar gait. Each frequency on its own sprocket chain
- Arc Line: 4 sprocket sizes (20,21,27,35) → 4 incommensurate frequencies → 27-min period before exact repeat

### 3. Phase Offset (Position Along Shaft)
**Physical implementation**: Objects at different positions along a rotating shaft are at different phases of oscillation.
- Every helix-driven sculpture uses this
- Square Wave: 9 discs offset 45° each → traveling wave illusion from single rotation
- Triple Helix: 37 sliders distributed along each helix → continuous phase gradient

### 4. Variable Amplitude (Adjustable Eccentricity)
**Physical implementation**: Sliding a disc along its shaft changes the offset from center.
- Square Wave: plywood discs can be repositioned on shaft
- This is how Margolin "tunes" the wave — adjusting each cam's eccentricity during installation

### 5. Topological Wave Mapping
**Physical implementation**: Map a wave equation onto a non-Euclidean surface.
- Mobius Wave: 3.5 wavelengths ensures continuity across the twist
- Trefoil: 3 cables trace a trefoil knot (hypotrochoid on torus)
- Dandelion Wave: 132 geodesic points on a sphere

### 6. String Path Optimization (Combinatorial)
**Physical implementation**: For n pulleys, 2ⁿ possible string routings. Only the shortest path is physically correct (string finds minimum).
- Confluence (2024): Python + matplotlib to compute optimal paths through complex pulley matrices
- This is Margolin's most computational recent work — the design uses software, but the mechanism remains purely analog
- **Note**: Margolin's Python solver for string paths is NOT published. The algorithm is proprietary to his studio workflow

### 7. Imaginary Numbers Made Visible
**Physical implementation**: Planar 2D motion that the brain perceives as 3D.
- Arc Line: Each ring swings in a flat plane, but the chain of 20 rings, phase-offset, appears to weave through 3D space
- The brain assumes minimum curvature → infers a depth dimension that doesn't physically exist
- That inferred depth IS the imaginary component of a complex sinusoid: `z(t) = A·e^(iωt) = A·cos(ωt) + i·A·sin(ωt)`
- We see the real part. We hallucinate the imaginary part. The sculpture makes this visible.

### 8. Friction Cascade
**Physical implementation**: After n pulleys in series, friction compounds exponentially.
- `F_output ≈ F_input · μⁿ` where μ ≈ 0.95 per pulley
- After 9 pulleys: efficiency ≈ 0.95⁹ ≈ 0.63 (37% lost)
- After 20 pulleys: efficiency ≈ 0.95²⁰ ≈ 0.36 (64% lost)
- Magic Wave: 3000 pulleys total, but max ~9 in any single string path
- Design constraint: keep serial pulley count LOW, parallelize instead

## Design Process (Margolin's Workflow)
1. **Start with a feeling or observation** — caterpillar motion, wave on water, Mobius strip topology
2. **Find the mathematics** — wave equation, Fourier decomposition, parametric curves
3. **Choose a mechanism** — cam, helix, sprocket chain, eccentric
4. **Solve the string routing** — how to get motion from mechanism to every output point
5. **Build a model** — small scale, test the motion
6. **Scale up** — solve structural, friction, material challenges
7. **Tune** — adjust cam eccentricities, cable tensions, motor speed
8. **Install** — often site-specific, designed for the architecture

### Margolin Wave Ring — Our OpenSCAD Recreation
**File:** `margolin_wave_ring_v1.scad` (created Feb 2026, untested)
- **Concept:** Tilted ring wave driver — 60mm ring, 15° tilt, 9×9 grid (77 active rods)
- **Includes:** Pulleys, routed strings, wire mesh weave

**Central Formula (PRESERVE):**
```
rod_height(x, y, theta) = r * tan(tilt_deg) * sin(phi - theta_deg)
where:
  r = sqrt(x² + y²)        — distance from center
  phi = atan2(y, x)         — angle from center
  theta = ring rotation     — current rotation angle
```

**Bug found & fixed:** Sign error in formula — `sin(theta - phi)` corrected to `sin(phi - theta)`
**Known issue:** `tan(tilt)` vs `sin(tilt)` mismatch (~3.5% error at 15°, accepted for prototype)
