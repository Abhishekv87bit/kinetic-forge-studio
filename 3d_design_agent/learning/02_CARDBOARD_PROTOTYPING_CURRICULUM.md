# Cardboard Prototyping Curriculum
## 8 Weeks to Mechanism Mastery Through Your Hands

**Philosophy:** Cardboard before CAD. Always.

📖 **Key References:**
- Robert Addams, "Automata Design" p.64-69 (young children cardboard projects)
- Robert Addams p.67-69 (drinks carton cranks, lolly stick gears)
- Making Things Move p.264-265 (cardboard pantograph project)

**Why Cardboard?**
- Iterations take minutes, not hours
- Feel where mechanisms bind
- Understand Grashof by touch, not formula
- Mistakes cost cents, not time
- 10 cardboard iterations = 1 informed CAD design

# PRE-WEEK: AGE 4-5 EXPLORATION

**Purpose:** Build intuition before formal mechanisms. Great for teaching your daughter!

## Exercise Set A: Just Pivots (Age 4)

**Learning:** "One point = spin, two points = swing, three points = stuck!"

### Exercise A1: Single Pivot Spinner
1. Cut any shape (star, heart, circle)
2. Put brass fastener through center
3. Attach to cardboard base
4. **Observe:** It spins freely around ONE point

### Exercise A2: Two-Pivot Pendulum
1. Cut a strip with hole at top
2. Attach to base with fastener
3. **Observe:** It swings, but can't spin

### Exercise A3: Three-Pivot Triangle
1. Cut triangle with holes at corners
2. Pin all three to base
3. **Observe:** It can't move at all! It's RIGID.

**Key Learning:** Triangles are stable. That's why bridges use them!

## Exercise Set B: Levers (Age 4-5)

📖 **See:** Robert Addams p.45-47 for lever class diagrams

### Exercise B1: Balance Beam (First-Class Lever)
**Like:** See-saw
```
     LOAD        EFFORT
       ↓           ↓
   ────●───────────●────
              △
           FULCRUM
```
- Fulcrum in the middle
- Push down on one side, other goes up
- **Try:** Move fulcrum closer to load - what happens?

### Exercise B2: Catapult (Third-Class Lever)
**Like:** Throwing arm
```
     ────────●──► LOAD (projectile)
           ↑
        EFFORT
         ↑
    ●────┘
  FULCRUM
```
- Fulcrum at one end
- Effort in the middle
- Load at far end
- Small effort movement = BIG load movement

### Exercise B3: Nutcracker (Second-Class Lever)
**Like:** Wheelbarrow
```
     EFFORT
        ↓
   ────●────────●────
              ↑   △
            LOAD  FULCRUM
```
- Fulcrum at one end
- Load in the middle
- **Result:** Amplified force (mechanical advantage!)

# WEEK 1: Simple Crank
## Converting Rotation to Linear Motion

📖 **Book References:**
- Robert Addams p.16-20 (crank chapter)
- Robert Addams p.68 (drinks carton crank - fish, eel, bed bug examples!)
- Making Things Move p.258-259 (crank mechanics)

### Learning Goal
Understand how a rotating crank creates reciprocating motion.

### Build This
```
         OUTPUT
           │
           │ ← Connecting rod
           │
       ────●──── ← Crank disk with offset hole
           │
        HANDLE
```

### Exercise 1: Basic Crank (1 hr)
1. Cut a crank disk (5cm diameter)
2. Put center hole for shaft
3. Put offset hole (1.5cm from center) for connecting rod
4. Attach connecting rod
5. **Observe:** Output travels 2× the offset distance

**Key Learning:** Offset distance = half the stroke length

### Exercise 2: Vary the Offset (1 hr)
Make 3 cranks with different offsets:
- 1cm offset → 2cm stroke
- 2cm offset → 4cm stroke
- 3cm offset → 6cm stroke

**Feel it:** Larger offset = more motion but more resistance

### Exercise 3: Quick Return (1-2 hrs)
Add a second crank at 90° to the first.
- **Observe:** Smoother motion, no dead point stall

### Exercise 4: Fun Projects from Robert Addams (1-2 hrs)
📖 **See:** Robert Addams p.68

Build one of these drinks carton crank toys:
- **Origami fish** - Fish body on crank, swims back and forth
- **Wiggling eel** - Multiple segments on offset cranks
- **Bed bug** - Legs move when you turn handle

These show cranks in action with playful outputs!

### Exercise 5: Robot Walker (Optional)
📖 **See:** Robert Addams p.69

Simple walking figure using crank to legs linkage.

### Checkpoint Questions
- [ ] What determines stroke length?
- [ ] Why does a single crank sometimes stall?
- [ ] How does adding a second offset crank help?

# WEEK 2: Crank-Slider
## Piston Motion and Connecting Rod Angles

📖 **Book References:**
- Making Things Move p.258-259 (slider-crank mechanics)
- Making Things Move p.247 (scotch yoke comparison)

### Learning Goal
Understand how rod length affects motion quality.

### Build This
```
    ═══════●═══════► ← Slider (constrained to line)
              ╲
               ╲ ← Connecting rod
                ╲
                 ●  ← Crank
                 │
              HANDLE
```

### Exercise 1: Basic Slider-Crank (1 hr)
1. Create slider track (two parallel strips)
2. Make slider that moves smoothly in track
3. Attach connecting rod from slider to crank
4. **Test:** Does it move smoothly?

### Exercise 2: Rod Length Experiment (1.5 hrs)
Make 3 versions with different rod lengths:
- Short rod (just barely reaches) - FEEL THE BIND
- Medium rod (2× crank radius) - OK
- Long rod (4× crank radius) - SMOOTH

**Key Learning:** Longer rod = straighter motion, less side force

### Exercise 3: Scotch Yoke (1 hr)
Alternative: Slot in slider, pin on crank
- **Compare:** Perfectly sinusoidal vs. slider-crank's slight asymmetry

### Checkpoint Questions
- [ ] Why do short connecting rods feel jerky?
- [ ] What's the minimum rod length for smooth operation?
- [ ] When would you use scotch yoke vs. slider-crank?

# WEEK 3: Four-Bar Linkage
## The Grashof Condition by Feel

📖 **Book References:**
- Making Things Move p.262-265 (four-bar linkages, pantograph)
- Robert Addams p.39-41 (linkages chapter)
- Animal joints paper p.1-15 (bio-inspired four-bars!)

### Learning Goal
Understand why some linkages rotate fully and others oscillate.

### Build This (Three Times)
```
      ●───────────●  ← Coupler
     ╱             ╲
    ●               ●
    │               │
    │     GROUND    │
    ●───────────────●
  CRANK           ROCKER
```

### Exercise 1: Crank-Rocker (1.5 hrs)
Cut links with these proportions:
- Ground: 100mm
- Crank: 30mm (SHORTEST)
- Coupler: 80mm
- Rocker: 70mm

**Check:** 30 + 100 = 130 < 80 + 70 = 150 ✓ Grashof satisfied!
**Result:** Crank rotates 360°, rocker oscillates

### Exercise 2: Double-Rocker (1.5 hrs)
Cut links with these proportions:
- Ground: 100mm
- Input: 60mm
- Coupler: 50mm
- Output: 70mm

**Check:** 50 + 100 = 150 > 60 + 70 = 130 ✗ Grashof violated!
**Result:** BOTH sides only oscillate - feel the limits!

### Exercise 3: Parallel Four-Bar (1 hr)
- All links equal: 50mm each
- Opposite links parallel
**Result:** Coupler stays level as it moves

### Exercise 4: Coupler Curve (1 hr)
Add a pointer extending from the coupler.
- Trace its path with pencil
- **See:** The "secret" curve that Pyslvs synthesizes!

### Checkpoint Questions
- [ ] Can you feel when a linkage "wants" to move vs. binds?
- [ ] What happens at the Grashof boundary?
- [ ] Why do some motions feel smooth and others jerky?

# WEEK 4: Cams
## Shape Determines Motion

📖 **Book References:**
- Robert Addams p.8-15 (complete cam chapter with all profile diagrams!)
- Robert Addams p.69 (owl with cam - full cardboard example!)
- Making Things Move p.240-242 (edge, disk, drum cams)

### Learning Goal
Understand how cam profile creates follower motion.

### Build This
```
         ↑
         │ ← Follower (up/down)
         ▼
     ╭─────╮
    ╱       ╲ ← Cam (rotating shape)
   │    ●    │
    ╲       ╱
     ╰─────╯
         │
      HANDLE
```

### Exercise 1: Circle (Eccentric) Cam (1 hr)
1. Cut perfect circle (5cm diameter)
2. Put shaft hole OFF CENTER (1cm from center)
3. Add follower that rides on edge
4. **Result:** Sine wave motion, smooth but simple

### Exercise 2: Egg Cam (1 hr)
1. Cut egg shape (wider at one end)
2. Center hole at narrow end
3. **Result:** Slow rise, quick drop (or vice versa)

**Key Learning:** Asymmetric shape = asymmetric motion

### Exercise 3: Heart Cam (1 hr)
1. Cut heart shape
2. Center hole at indent
3. **Result:** Two rises per rotation!

### Exercise 4: Dwell Cam (1.5 hrs)
1. Cut circle with one flat edge
2. **Result:** Motion, then PAUSE at flat!
3. Try: Rise-Dwell-Fall-Dwell (two flats)

### Checkpoint Questions
- [ ] How does the slope of the cam edge affect speed?
- [ ] What shape gives the longest dwell?
- [ ] Why might a heart cam be useful?

# WEEK 5: Gear Train
## Teeth, Mesh, and Ratio

📖 **Book References:**
- Robert Addams p.21-30 (complete gear chapter)
- Robert Addams p.69 (lolly stick gears for kids!)
- Making Things Move p.231-245 (gear mechanics, types)

### Learning Goal
Understand gear ratios and feel backlash.

### Build This
```
     ●═══════●═══════●
   GEAR 1   GEAR 2   GEAR 3
   (large)  (medium) (small)
```

### Exercise 1: Simple Gear Pair (1.5 hrs)
1. Cut two gears from foam board
   - Gear A: 40 teeth (large circle, cut triangles)
   - Gear B: 20 teeth (half the size)
2. Mount on cardboard base with proper spacing
3. **Test:** Turn A once, B turns twice!

**Key Learning:** Teeth ratio = speed ratio

### Exercise 2: Feel Backlash (1 hr)
1. Separate gears slightly
2. Turn one - notice the "slop" before the other moves
3. Push gears closer - less backlash but more friction
4. **Balance:** Find the sweet spot

### Exercise 3: Compound Train (1.5 hrs)
1. Make 3-gear train
2. Gear A: 40T → Gear B: 20T → Gear C: 40T → Gear D: 20T
3. **Calculate:** 40/20 × 40/20 = 4:1 ratio
4. **Verify:** Input 1 rotation, output 4 rotations!

### Exercise 4: Rack and Pinion (1 hr)
1. Cut straight rack (flat gear)
2. Cut pinion (circular gear)
3. **Result:** Rotation → Linear motion (alternative to crank!)

### Exercise 5: Lolly Stick Gears (Great with Daughter!)
📖 **See:** Robert Addams p.69

1. Use wooden lolly sticks as gear teeth
2. Arrange in spoke pattern on cardboard disks
3. Mesh two "gears" together
4. **Result:** Quick, cheap gear prototypes!

### Exercise 6: Friction Wheels (No Teeth!)
1. Two cardboard circles in contact
2. Press together, turn one
3. **Observe:** Other turns opposite direction
4. **Note:** Slips under load - that's why we need teeth!

### Exercise 7: Pulley & Belt
📖 **See:** Robert Addams p.35-38

1. Two pulleys (different sizes)
2. Rubber band as belt
3. **Observe:** Size difference = speed difference

### Checkpoint Questions
- [ ] What gear ratio would you need for 60:1 reduction?
- [ ] Why is some backlash actually necessary?
- [ ] When would you use rack and pinion vs. crank-slider?

# WEEK 6: Parallel Motion
## Straight-Line Mechanisms

📖 **Book References:**
- Making Things Move p.264-265 (cardboard pantograph project!)
- Robert Addams p.39-41 (linkages)

### Learning Goal
Create (approximately) straight motion from rotary input.

### Build This
```
        POINT TRACES
       STRAIGHT LINE
             │
             ↓
     ●───────●───────●
    ╱         │       ╲
   ●          │        ●
   │        ●─┴─●      │
   │       COUPLER     │
   ●──────────────────●
         GROUND
```

### Exercise 1: Watt's Linkage (1.5 hrs)
Historical! James Watt invented this for steam engines.
1. Two rockers, one coupler connecting their ends
2. Midpoint of coupler traces approximately straight line
3. **Observe:** Not perfect, but very close!

### Exercise 2: Chebyshev Linkage (1.5 hrs)
Specific proportions for optimal straightness:
- Ground: 2 units
- Cranks: 2.5 units each
- Coupler: 1 unit
- **Result:** Better than Watt for straight motion!

### Exercise 3: Peaucellier Linkage (1.5 hrs)
True straight line (but complex):
- 8 links total
- Uses the "inverse" property
- **Result:** Mathematically perfect straight line!

### Checkpoint Questions
- [ ] Why is straight-line motion hard to achieve with linkages?
- [ ] When is "approximately straight" good enough?
- [ ] What's the simplest way to get true straight motion?

# WEEK 7: Integration
## Combining Mechanisms

### Learning Goal
Connect multiple mechanism types to create complex motion.

### Exercise 1: Motor → Cam → Linkage (2 hrs)
1. Crank input
2. Drives cam
3. Cam drives four-bar follower
4. **Result:** Custom motion profile!

### Exercise 2: Gear Reduction → Crank (2 hrs)
1. Input crank turns fast
2. Gear train slows it down (4:1)
3. Output crank at slow speed
4. **Result:** Slow, powerful motion

### Exercise 3: Phase-Coupled Outputs (2 hrs)
1. Single input crank
2. Multiple output cranks at different phases
3. **Result:** Wave motion (like your Ocean Waves project!)

### Design Your Own (1-2 hrs)
Combine at least 2 mechanism types:
- What input? (hand crank, motor, weight)
- What transformation? (speed, direction, motion type)
- What output? (oscillation, rotation, linear)

### Checkpoint Questions
- [ ] What determines the order of mechanisms in a chain?
- [ ] How do you manage tolerance stack-up?
- [ ] When would you use parallel paths vs. serial?

# WEEK 8: Cardboard → CAD
## Transferring to Digital and Physical

### Learning Goal
Take working cardboard prototype to OpenSCAD/3D print.

### Exercise 1: Measure Everything (1 hr)
For your best cardboard mechanism:
- Link lengths (measure pivot-to-pivot)
- Shaft diameters
- Clearances that work
- What binds, what's loose

### Exercise 2: Document the Mechanism (1 hr)
Create a sketch with:
- All dimensions labeled
- Joint types marked
- Range of motion noted
- Problem areas identified

### Exercise 3: OpenSCAD Translation (2 hrs)
Convert to parametric code:
```openscad
// Example: Four-bar from cardboard
ground = 100;  // Measured: 100mm
crank = 30;    // Measured: 30mm
coupler = 80;  // Measured: 80mm
rocker = 70;   // Measured: 70mm
// ... continue building
```

### Exercise 4: First Print (1-2 hrs)
Print ONE link first:
- Check dimensions
- Test fit with cardboard version
- Adjust tolerances
- Then print the rest

### The Bridge Checklist
Before leaving cardboard:
- [ ] Mechanism works smoothly through full range?
- [ ] All measurements documented?
- [ ] Binding points identified and solved?
- [ ] Clearances noted (what works)?
- [ ] Ready to recreate in CAD?

## Remember

> "Cardboard before CAD" isn't just about speed - it's about building intuition that no simulation can give you.

When something binds in cardboard, you FEEL it.
When proportions are off, you SEE it.
When the motion is right, you KNOW it.

That embodied knowledge transfers directly to better CAD designs.

**The goal isn't beautiful cardboard models - it's ugly prototypes that WORK.**
