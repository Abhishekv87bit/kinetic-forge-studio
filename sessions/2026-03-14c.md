## What Was Done

### GAP-PPL-009: Caching Layer
- **TTLCache** (`app/middleware/cache.py`): Async-safe in-memory cache with LRU eviction, TTL expiry, hit/miss stats
  - 3 pre-configured instances: `prompt_cache` (32, no TTL), `search_cache` (128, 1hr), `execution_cache` (64, no TTL)
  - Global `get_cache_stats()` and `clear_all_caches()` helpers
  - Project-specific invalidation via `clear_project_cache()`
- **Anthropic prompt caching**: Changed `_call_claude()` system field from plain string to content block with `cache_control: {"type": "ephemeral"}` — ~90% cost reduction on repeated system prompts
- **Health wiring**: Cache stats added to `/api/health` response
- **13 unit tests** + 1 Anthropic cache test + 1 health test = 15 tests, all passing
